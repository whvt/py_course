Flow Control. Loops.¶

Введение в условные операторы¶

Перед тем как погрузиться в детали, давайте определим, что такое условный оператор и зачем он нам нужен. В программировании, условный оператор позволяет выполнить определенный блок кода только при выполнении определенного условия. Это позволяет программе принимать решения на основе данных или внешних факторов.

Операторы сравнения¶

Операторы сравнения являются ключевым инструментом для проверки условий. Они позволяют нам сравнивать значения и возвращать результаты в виде логического значения True или False. Давайте рассмотрим некоторые наиболее часто используемые операторы сравнения, такие как равно (==), не равно (!=), больше (>), меньше (<), больше или равно (>=), меньше или равно (<=).

Проверка истинности в Python¶

  • Любое число, не равное 0, или непустой объект - истина.
  • Числа, равные 0, пустые объекты и значение None - ложь
  • Операции сравнения применяются к структурам данных рекурсивно
  • Операции сравнения возвращают True или False
  • Логические операторы and и or возвращают истинный или ложный объект-операнд

if-else¶

Оператор else позволяет задать блок кода, который выполнится, если условие оператора if не истинно. Оператор elif (сокращение от "else if") используется для проверки дополнительных условий, если предыдущие условия были ложными. Давайте рассмотрим пример:

if x > 0:
    print("Число положительное")
elif x < 0:
    print("Число отрицательное")
else:
    print("Число равно нулю")

В языке Python такого же эффекта можно добиться при помощи выражения – благодаря тому, что функция bool преобразует X в соответствующее целое число 1 или 0, которое затем используется для выбора требуемого значения из списка:

A = [“Z”, “Y”][bool(“X”)]

Циклы¶

Циклы - это инструмент, который позволяет выполнять определенные действия множество раз. Они особенно полезны, когда нам нужно повторять один и тот же код или операции несколько раз. https://pythontutor.com/render.html#mode=edit

В Python существует два основных типа циклов:

  • for
  • while.

Цикл for¶

Цикл for позволяет перебирать элементы в заданном диапазоне или в коллекции. Он выполняет указанный блок кода для каждого элемента в заданной последовательности.

In [43]:
for i in range(5):
    print(i)

T-e-s-t-.-s-t-r-i-n-g-

В этом примере мы используем цикл for для вывода чисел от 0 до 4. Функция range(5) создает последовательность чисел от 0 до 4, и каждое число передается в переменную i, которую мы используем внутри цикла.

Циклы позволяют автоматизировать повторяющиеся задачи и обрабатывать большие объемы данных. Они особенно полезны при работе с коллекциями, такими как списки или словари.

Оператор break¶

Оператор break - это инструкция, которая позволяет прервать выполнение цикла досрочно. Когда оператор break достигается внутри цикла, выполнение цикла немедленно прекращается, и управление передается к коду, следующему после цикла.

In [62]:
for i in range(1, 11):
    if i == 5:
        break
    print(i)

1
2
3
4

В этом примере мы используем цикл for для итерации от 1 до 10. Но при условии, что i равно 5, мы вставляем оператор break, который прерывает выполнение цикла. В результате на экране будут выведены числа 1, 2, 3, и 4. Как только достигнуто условие i == 5, цикл прекращается и код, следующий за циклом, начинает выполняться.

Оператор break может быть полезен, когда необходимо прервать цикл, когда выполняется определенное условие или достигается определенный результат. Он позволяет сэкономить время и ресурсы, прерывая выполнение цикла, когда это необходимо.

Важно помнить, что оператор break прерывает только ближайший внутренний цикл, в котором он находится. Если в программе есть вложенные циклы, break остановит только самый внутренний цикл.

Оператор continue¶

Оператор continue - это инструкция, которая прерывает текущую итерацию цикла и переходит к следующей итерации. Когда оператор continue достигается внутри цикла, оставшаяся часть кода внутри цикла не выполняется, и управление возвращается в начало цикла для следующей итерации.

In [59]:
for i in range(1, 11):
    if i == 5:
        continue
    print(i)

В этом примере мы используем цикл for для итерации от 1 до 10. Но при условии, что i равно 5, мы вставляем оператор continue, который пропускает оставшуюся часть кода внутри цикла и переходит к следующей итерации. В результате на экране будут выведены числа от 1 до 4 и от 6 до 10.

Оператор continue может быть полезен, когда нужно пропустить определенные итерации цикла на основе условия. Например, можно использовать его для пропуска обработки определенных значений или выполнения определенных операций для определенных условий.

Важно помнить, что оператор continue прерывает только текущую итерацию цикла и переходит к следующей итерации. Весь оставшийся код внутри цикла после оператора continue не выполняется. Если в программе есть вложенные циклы, continue пропустит только текущую итерацию внутреннего цикла, но продолжит выполнение остальных итераций внешнего цикла.

Механизм для проверки данных, их распаковки и управления потоком выполнения. Сопоставление выражения с некоторым шаблономб в каком-то смысле похоже на if/else/elif конструкцию:

status = response.status
if status == 200:
    do_something(response.data)  # OK
elif status == 301 or status == 302:
    retry(response.location)  # Redirect
elif status == 401:
    retry(auth=get_credentials())  # Login first
elif status == 426:
    sleep(DELAY)  # Server is swamped, try after a bit
    retry()
else:
    raise RequestError("we couldn't get the data")
